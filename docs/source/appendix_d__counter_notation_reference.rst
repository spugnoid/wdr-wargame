Appendix D — Counter Notation Reference
=======================================

Counter Face Layout
-------------------


.. list-table::
   :header-rows: 1
   :widths: auto

   * - **Position**
     - **Field**
     - **Example**
   * - Upper left
     - Unit type symbol
     - [X] = rifle squad
   * - Upper centre
     - Unit ID
     - GREN 43 = Wehrmacht Grenadier, 1943
   * - Upper right
     - Action values
     - M1 F2 G3
   * - Centre lines
     - Fire lines (1–3)
     - ─● 7 ⬡4 -1
   * - Lower left
     - Defence value
     - 8
   * - Lower right
     - Morale value
     - 5
   * - Bottom band
     - Support weapon (if present)
     - MG42b 7 ⬡4 -1


Fire Line Notation
------------------


.. list-table::
   :header-rows: 1
   :widths: auto

   * - **Element**
     - **Meaning**
     - **Example**
   * - Icon
     - Weapon class (see Section 3.4)
     - ─● = LMG
   * - rFP
     - Base firepower at range 0–1
     - 7
   * - ⬡h
     - Hex interval between FP reductions
     - ⬡4 = every 4 hexes
   * - -f
     - FP lost per interval
     - -1 = lose 1 FP per interval
